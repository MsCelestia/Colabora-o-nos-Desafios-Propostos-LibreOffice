# Guia Completo: Macro de Conversão RP → R$
Transforme Riot Points em Reais automaticamente no LibreOffice Calc!

## O que é isso?
Sabe quando você precisa calcular quanto vale uma conta de League of Legends em reais? Ou descobrir quantos RP você consegue com determinado valor?
Esta macro faz isso automaticamente para você!
Basta digitar uma fórmula simples no Calc e pronto: a conversão é feita na hora! 

## Instalando a Macro (SUPER FÁCIL!)
### Opção A: Jeito Mais Simples (Recomendado)
1. Baixe o arquivo conversao_rp.bas que está junto com este guia
2. Abra o LibreOffice Calc (aquele programa de planilhas)
3. No menu superior, clique em:
Ferramentas → Macros → Organizar Macros → LibreOffice Basic
4. Na janela que abrir, clique no botão "Importar..." (fica embaixo, no canto)
5. Procure e selecione o arquivo conversao_rp.bas que você baixou
6. Uma janelinha vai perguntar se quer importar. Clique em "OK"
7. Pronto! Pode fechar tudo. A macro já está instalada! 

### Opção B: Copiando o Código Manualmente
Se a Opção A não funcionou, tente esta:
1. Abra o LibreOffice Calc
2. Pressione as teclas Alt + F11 juntas (isso abre o editor de código)
3. No painel da esquerda, você verá "Standard". Clique com o botão direito nele
4. Escolha "Inserir" → "Módulo"
5. Abra o arquivo conversao_rp.bas em um bloco de notas qualquer
6. Selecione TODO o conteúdo (Ctrl + A) e copie (Ctrl + C)
7. Volte para o LibreOffice e cole (Ctrl + V) dentro daquele módulo que você criou
8. Aperte Ctrl + S para salvar
9. Feche o editor (pode fechar o "X" mesmo)
10. Pronto! Macro instalada!
