# Propostas de melhorias para a interface e experiência do usuário:

## Objetivo

- Propor melhorias de experiência do usuário (UX) para o LibreOffice Calc, focadas em usuários iniciantes e intermediários, com base em reclamações reais da internet (fóruns, Reddit, StackOverflow e comunidades de software livre).

### Melhorias propostas:
### 1. Digitação de Fórmulas

- Problema atual: interface fria, tooltips técnicos, mensagens de erro crípticas (#NOME?, Erro 504), difícil entendimento de argumentos.

- Proposta: tooltips expandidos e humanizados, destaque visual dos argumentos, mini-preview do resultado esperado.

#### 2. Construtor de Fórmulas

- Problema atual: muitas janelas, campos técnicos, pouca orientação.

- Proposta: etapas guiadas passo a passo, exemplos automáticos conforme o usuário preenche, validação visual (verde = ok, vermelho = erro), botão “ver exemplo aplicado na planilha”.

#### 3. Mensagens de Erro

- Problema atual: erros criptográficos, sem explicação clara.

- Proposta: mensagens humanas (“Você digitou o nome da função errado”), sugestões automáticas (“Você quis dizer PROCV?”), link rápido para ajuda contextual.

#### 4. Acessibilidade

- Alto contraste opcional

- Ajuste de tamanho do tooltip

- Compatível com leitores de tela

- Ícones + texto (não só cor)

#### 5. Ajuda Contextual Inteligente

- Dicas curtas aparecem quando o usuário comete erros

- Não substitui a documentação, mas ajuda no contexto

#### 6. Modo Iniciante / Avançado

- Interface simplificada para iniciantes

- Funções básicas destacadas

- Usuário pode alternar quando quiser

#### 7. Exemplos Prontos por Categoria

- Financeiro, escolar, jogos, estatística

- Permite aplicar funções em casos da vida real, reduzindo confusão
